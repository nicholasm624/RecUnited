## FAQ

- **How can I download it?**

RecUnited is currently in the development phase and is not yet available for download. We're working hard to bring you something great — stay tuned for our planned release in late May 2026!

- **Is RecUnited Legal?**

RecUnited is a fan-made preservation project. We don't distribute copyrighted game files — you need to already have the game installed. We operate legally under fair use principles like many other private servers.

- **Does It Cost any Real Money?**

No. The server is completely free to play. We cover backend server costs through community donations. We will never sell in-game items.

- **Can i get my old rooms and stuff back?**

Sadly, no. We cannot access your old data from their servers. Everyone gets to start totally fresh!

- **Which platforms our supported?**

We are focusing on PC and PCVR first, with Meta Quest sideloading following soon after. We don't plan to release on consoles simply due to how locked-down those stores are.

- **How do we plan to have users install the client?**

We plan to use a client pacher which you will be able to use to patch your PRE DOWNLOADED RecRoom Client.

- **How can i report a bug?**

Go to the #🤖│bot-commands channel [on our discord]([(https://discord.gg/JW5jjkjCuj)] and type /bug, then fill out the quick form.

- **Can I create custom rooms?**

We plan to implment the Maker Pen into RecUnited so you can make with your friends and save your rooms to our server

- **How can i help out?**

Join the [Discord](https://discord.gg/zZQrYBmjwv) , contribute to our GitHub repo, or help test out the network code when we do live lobby tests!

*Have a question that isn't answered here? Feel free to [ask on our discord](https://discord.gg/zZQrYBmjwv) and we'll get back to you.*
